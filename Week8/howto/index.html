What's this all about?

This site is a brief tutorial for how to use The Guardian's totally free, totally awesome news API -- what they call the Open Platform. Since there's some pretty solid documentation, but not much of a Getting Started tutorial - hopefully this will bridge that gap for you. The intended audience are beginner (or beginner-intermediate) programmers who have seen "API" all over the place and either aren't sure what it is or how to use it. While we will cover the Guardian's news API in particular, the concepts introduced can be applied generally (more or less).

OK. So now what?

Great question. Here's the plan:

  1. Explain what an API is
  2. Show how to get data from an API
  3. Discuss how to use said data
  4. Build a cool scrolling news ticker like the kind you see on CNN
